# 알파 탐색 프로젝트에서 각 지표의 역할

## 1. 매크로 지표

> 지금은 게임을 해도 되는 국면인가?

매크로 = 언제 싸울지 결정하는 신호

**역할**
- 시장 노출(exposure) 결정
- 알파의 작동 가능 환경 판단

**성격**
- 느림 (월~분기)
- 방향성 및 국면 판단
- 알파 생성 보다는 리스크 관리 용도

## 2. 기본적 분석 지표

> 어떤 종목에서 알파를 찾을 것인가?

기본적 = 알파가 잘 나오는 종목만 남기는 필터

**역할**
- 유니버스 정의
- 구조적 리스크 제거
- 알파 증폭 및 안정화

**성격**
- 느림 (분기)
- 퀄리티 및 지속성 중심
- 알파의 직접 생성 보다는 알파를 만드는 환경 제공

## 3. 기술적 분석 지표

> 지금 들어가고, 언제 나올 것인가?

기술적 = 실제로 돈을 벌거나 읽게 만드는 신호

**역할**
- 진입/청산 타이밍
- 실제 수익률을 만드는 주체

**성격**
- 빠름 (일~주)
- 확률적이며 노이즈 많음
- 직접적인 알파 생성
