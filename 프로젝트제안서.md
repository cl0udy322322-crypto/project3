# 프로젝트 제안서

## 1. 프로젝트 제목

알파 탐색을 위한 투자 인디케이터 개발


## 2. 프로젝트 내용

본 프로젝트는  
- 매크로, 펀더멘털, 기술적 지표를 결합한 투자 인디케이터를 설계하고
- KOSPI200 및 KOSDAQ150 종목을 투자 유니버스로 설정하여
- 알파(초과수익)를 창출할 수 있는 투자 전략의 가능성을 탐색하고
- 백테스트를 통해 이를 검증하는 것을 목표로 한다.


## 3. 최종 목표 및 결과물

- 매크로·펀더멘털·기술적 기반 커스텀 인디케이터 개발
- KOSPI200 / KOSDAQ150 기반 멀티 팩터 스코어링 모델 구축
- 알파(초과수익) 존재 여부에 대한 정량적 검증
- Buy & Hold 및 벤치마크 대비 초과수익 여부 분석


## 4. 프로젝트의 의미

- 기술적 분석 중심의 단순 매매 전략을 넘어  
  매크로·펀더멘털·기술적 분석을 통합한 멀티 팩터 접근 방식 적용  
- 데이터 기반 분석을 통해 알파의 존재 여부를 검증하는  
  체계적인 투자 전략 설계 및 검증 프로세스 학습


## 5. 프로젝트의 차별성, 독창성

- 전략 구현이 아닌 알파 탐색 및 검증에 초점
- 한국 시장(KOSPI200 / KOSDAQ150)에 특화된 팩터 분석


## 6. 필요한 데이터 소스 및 저장장소 및 데이터별 확보 방법

- 재무 데이터: DART, KRX (분기 재무제표 활용)
- 주가 데이터: Financial Data Reader(FDR), 증권사 API
- 거시 데이터: FRED
- 데이터 저장 방식: CSV 파일, Python DataFrame
- 저장소 관리: GitHub Repository


## 7. 분석 내용 및 적용 기술 (단계별)

1. 데이터 수집 및 전처리
2. 매크로·기본적·기술적 팩터 정의
3. 종목 필터링 및 스코어링 모델 구축
4. 포트폴리오 전략 설계
5. 백테스트 수행
6. 성과 및 리스크 분석
7. 결과 해석 및 문서화


## 8. 프로세스 및 일정

시장조사 → 주제설정 → 실행계획 수립 → 데이터 확보 → 데이터 전처리 →  
데이터 탐색 및 분석 → 최종 분석 결과 도출 → 결과 정리 →  
프로젝트 평가 → 발표자료 제작 → 발표
